# CVM++ 🚀

Custom Stack-Based Virtual Machine and Compiler built in C++.

---

# 📌 Overview

CVM++ is a lightweight scripting language runtime built completely from scratch.

The project demonstrates how source code transforms into executable bytecode through:

- Lexical Analysis
- Parsing
- AST Construction
- Bytecode Compilation
- Stack-Based Virtual Machine Execution

---

# 🧠 Features

- Custom Lexer
- Recursive Descent Parser
- AST Generation
- Bytecode Compiler
- Stack-Based Virtual Machine
- Arithmetic Operations
- Variable Support
- CLI Execution

---

# ⚙️ Architecture

Source Code
↓
